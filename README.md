# Network Troubleshooting Lab Portfolio

This repository contains hands-on troubleshooting labs completed using CertMaster Perform and Cisco Packet Tracer.  

The focus of this repository is to develop real-world troubleshooting skills by identifying, diagnosing, and resolving common networking issues in simulated environments.

---

## 🔧 Core Skills Demonstrated

- Network troubleshooting methodology
- IP configuration analysis (IP address, subnet mask, default gateway)
- VLAN troubleshooting and segmentation issues
- DHCP configuration and failure diagnosis
- Inter-VLAN routing and connectivity troubleshooting
- DNS resolution issues
- Use of diagnostic tools:
  - `ping`
  - `ipconfig`
  - `traceroute`
  - `show ip interface brief`
  - `show vlan brief`

---

## 🧠 Troubleshooting Approach

Each lab follows a structured methodology:

1. Identify the problem and symptoms  
2. Gather information and examine configurations  
3. Test connectivity using diagnostic tools  
4. Compare with known working systems when applicable  
5. Establish a probable cause  
6. Implement a solution  
7. Verify full system functionality
